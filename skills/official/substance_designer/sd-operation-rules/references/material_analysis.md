# 材质科学分析框架

## 三问推导法

对每个目标材质回答三个问题：
1. **原料是什么？** → 纤维/颗粒/晶体的基础形态和尺寸
2. **怎么加工的？** → 结构排列方式和表面状态
3. **经历了什么？** → 风化/磨损/老化的叠加层

## 原料→微观对照表

| 类别 | 原料差异 | 微观表现 |
|------|---------|---------|
| 布料 | 棉(粗糙吸色) vs 丝(光滑高光) vs 涤纶(均匀微塑) | 棉=绞合纤维+毛刺, 丝=极细光滑, 涤纶=均匀 |
| 混凝土 | 高沙(细腻) vs 碎石多(粗骨料外露) | 高沙=平滑微孔, 碎石=凸起+空洞 |
| 金属 | 铸铁(晶粒粗) vs 铝(细腻) vs 不锈钢(镜面) | 铸铁=砂眼, 铝=细划痕, 不锈钢=极细拉丝 |
| 木材 | 软木(年轮宽) vs 硬木(年轮密) | 软木=纤维粗导管, 硬木=纹理细 |

## 工艺→微观对照表

| 工艺 | 差异 | 表现 |
|------|------|------|
| 编织 | 平纹(1/1) vs **斜纹(3/1=牛仔!)** vs 缎纹 | 平纹=均匀, 斜纹=对角线, 缎纹=光滑 |
| 染色 | 纱线染色=经纬异色 vs 匹染=均匀 vs 印花=表面 | 纱线染色→经纬色差是核心 |
| 表面 | 抛光 vs 喷砂 vs 拉丝 vs 阳极氧化 | 完全不同的 Roughness 分布 |

## 尺度框架

| 场景 | 覆盖面积 | 像素密度 |
|------|---------|---------|
| 服装特写 | 10-15cm | 0.1-0.15mm/px |
| 标准面料 | 25-40cm | 0.25-0.4mm/px |
| 建筑表面 | 100-200cm | 1-2mm/px |

## 层级尺度排序（严格）

```
L4低频(30-100%面积) > L1主结构(2-10%) > L2次结构(0.5-2%) > L3微观(0.1-0.5%)
```

## 常见思维陷阱

- ⛔ Uniform纯色做纹理源 → warp后仍纯色
- ⛔ 先sharpen再blur → 放大CG感（应 blur→warp→sharpen）
- ⛔ 不声明覆盖面积就调scale → 盲调
- ⛔ 所有层同scale → 层次消失
- ⛔ 先着色后做Height → 颠倒（Height优先）
