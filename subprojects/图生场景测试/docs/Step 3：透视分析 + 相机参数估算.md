细化：
3.1 透视线检测与消失点计算
● 对于斜45°视角 low-poly 场景，通常有明显的建筑边缘线
● OpenCV Hough 变换检测直线 → 聚类求消失点
● 两个消失点 → 估算相机内参（焦距/FOV）
● 如果场景缺少明显直线（纯自然场景），退化为"基于已知物体尺寸反推"
3.2 地平面估算
● 斜45°视角的关键优势：地面投影关系相对规则
● 假设地面平坦，利用消失点 + 已知参考尺寸建立投影矩阵
● 单应性变换（Homography）：原图地面 → 俯视图
3.3 物体 3D 坐标估算
● 每个物体的 ground_contact_2d → 通过逆投影 → 地面上的 3D 坐标 (X, Y, 0)
● 物体高度：利用像素高度 + 距离 → 反算实际高度
● 输出每个物体的 position_3d: [X, Y, Z] 和 scale_3d: [W, D, H]

关键挑战：
● 单张图的深度估算天然有歧义 → 需要锚定参考物（选一个"已知尺寸"的物体作为标尺）
● 斜45°视角的好处是几何约束比较强，比自由视角好解很多
● Low-poly 建筑通常有明显的正交边缘，透视线检测友好
