## 一、插件入口与窗口管理

## 二、一体化 Dashboard 面板

### 2.1 状态栏（可折叠）
- 默认折叠，仅显示摘要
- 展开后: 版本号、服务器地址、统计信息、操作按钮组
- 状态颜色: 已连接=绿色，未连接=灰色
- 自动刷新: 每 2 秒轮询状态

### 2.2 消息区域（聊天历史）
- 容器: SScrollBox，纵向滚动
- 消息上限: 500 条
- 消息模型: Sender/Content/Timestamp/bIsCode
- 发送者颜色区分: user(蓝)/assistant(绿)/streaming(灰绿)/thinking(淡紫)/system(灰)
- 本地化标签

### 2.3 快捷输入分栏（可折叠）
- **功能说明**: 用户可自定义常用指令为快捷按钮，一键发送
- **操作**: 添加/编辑/删除快捷按钮
- **存储**: JSON 文件持久化保存配置

### 2.4 输入区域
- 多行输入框（最小 52px，最大 120px）
- 发送模式切换: Enter / Ctrl+Enter
- 动态占位提示文本
- / 前缀触发 Slash 命令菜单

### 2.5 底部工具栏
- **+ 新会话**: 创建全新对话会话，清空当前聊天历史
- **创建 Skill**: 启动 AI 对话式 Skill 创建流程，引导用户完成 Skill 定义
- **⚙ 管理**: 打开 MCP 与 Skill 管理面板（详见第十一章），可配置服务器状态、查看工具列表、管理 Skill 生命周期
- **语言切换按钮**: 实时切换 UI 中英文显示
- **发送模式复选框**: 切换 Enter 发送 / Ctrl+Enter 发送模式

## 三、Slash 快捷命令系统

### 3.1 命令菜单
- **触发方式**: 在输入框中输入 `/` 前缀自动弹出命令菜单
- **菜单位置**: 悬浮于输入框上方
- **实时过滤**: 根据输入内容动态过滤可用命令

### 3.2 命令列表
- **本地命令**:
  - `/connect` - 手动连接服务器
  - `/disconnect` - 断开服务器连接
  - `/diagnose` - 运行诊断检查
  - `/status` - 查看当前连接状态
  - `/clear` - 清空当前会话聊天记录
  - `/cancel` - 取消正在进行的请求
  - `/help` - 显示帮助信息
- **AI 命令**:
  - `/new` - 创建新会话
  - `/compact` - 压缩会话上下文
  - `/review` - 审阅当前操作
  - `/undo` - 撤销上一步操作

## 十二、Plan 模式

### 12.1 Plan 生成与展示
- **Plan 模式开关**: 用户可选择启用 Plan 模式，AI 会先生成执行计划而非直接执行
- **AI 生成 Plan**: AI 返回 JSON 格式的执行计划，包含步骤列表和依赖关系
- **步骤卡片**: 每个步骤以卡片形式展示，状态包括:
  - 待执行（Pending）
  - 执行中（In Progress）
  - 已完成（Completed）
  - 失败（Failed）
  - 已跳过（Skipped）

### 12.2 Plan 交互
- **审阅计划**: 用户可在执行前查看完整计划，确认步骤合理性
- **编辑计划**: 可删除尚未开始执行的步骤，调整执行顺序
- **执行计划**: 逐步骤执行，实时更新每个步骤的状态
- **暂停/继续**: 支持暂停执行流程，之后可继续从暂停处开始

### 12.3 执行控制
- **暂停全部**: 暂停所有待执行步骤
- **取消全部**: 中断整个计划执行
- **步骤失败处理**: 单步骤失败时提示用户，可选择重试、跳过或终止

## 十三、Skill 与 MCP 管理面板

### 13.1 Skill 管理
- **三层展示**: 按来源分类显示
  - official: 官方内置 Skill
  - marketplace: 社区共享 Skill
  - user: 用户自定义 Skill
- **操作**: 启用/禁用/钉选常用 Skill
- **筛选**: 按层级分类筛选 Skill 列表
- **详情**: 点击查看 Skill 详细信息弹窗

### 13.2 Skill 生命周期
- **创建**: 通过 AI 对话式引导创建新 Skill
- **编辑**: 修改现有 Skill 的配置和代码
- **发布**: 将 Skill 发布到 marketplace 或 official 频道
- **更新**: 检查并应用 Skill 更新
- **卸载/删除**: 移除不再需要的 Skill

### 13.3 MCP 管理
- **服务器状态**: 查看各 MCP 服务器的连接状态和运行情况
- **工具列表**: 浏览当前可用的 MCP 工具，了解工具功能和参数

### 13.4 面板入口
- Dashboard 底部工具栏 [⚙ 管理] 按钮，点击打开 Skill 与 MCP 管理面板
