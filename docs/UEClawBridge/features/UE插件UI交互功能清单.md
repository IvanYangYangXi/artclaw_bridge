## 二、一体化 Dashboard 面板

### 2.1 整体布局（自上而下）

```
┌──────────────────────────────────────────────────────┐
│ 状态栏 (可折叠)                                       │
│ [已连接] v0.1.0  ●...                     [设置]     │
├──────────────────────────────────────────────────────┤
│                                                      │
│                消息列表（聊天区域）                     │
│                (SScrollBox, FillHeight)               │
│                                                      │
├──────────────────────────────────────────────────────┤
│ ▶ 快捷输入                                 [+ 添加]  │
├──────────────────────────────────────────────────────┤
│ [会话▼] [+新会话] [Skills]    [Attach] [停止] [发送] │
│ ┌──────────────────────────────────────────────────┐ │
│ │ [IMG] clip.png 245KB [x]  [FILE] data.csv [x]   │ │  ← 附件预览栏（有附件时显示）
│ ├──────────────────────────────────────────────────┤ │
│ │ 输入框 (多行, 最小32px, 最大120px)               │ │
│ └──────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────┘
```

### 2.2 状态栏（可折叠）
- 默认折叠，仅显示摘要行
- 展开后: 版本号、服务器地址、统计信息、操作按钮组（连接/断开/诊断/日志/发送模式）
- 状态颜色: 已连接=绿色，未连接=灰色
- 自动刷新: 每 2 秒轮询状态
- 折叠行右侧按钮:
  - **设置**: 打开设置弹窗（语言切换、发送模式、静默模式、Plan 模式、Skills 管理）
  - **版本号**: 灰色小字

### 2.3 设置面板（弹窗）
- **入口**: 状态栏右侧 [设置] 按钮
- **窗口类型**: 独立 SWindow，可重复打开（已存在则激活）
- **设置项**:
  - **语言切换**: 显示目标语言名（中文→"English"，英文→"中文"），点击切换并刷新 UI
  - **发送模式**: 复选框，勾选 = Enter 发送，取消 = Ctrl+Enter 发送
  - **静默模式**: 两个按钮
    - 中风险静默: 文件修改/移动操作自动通过（绿色=开启）
    - 高风险静默: 文件删除/批量操作自动通过（红色=开启）
  - **Plan 模式**: 开启/关闭 Plan 模式（紫色=开启）
  - **Skills 管理**: 打开 Skill 与 MCP 管理面板
  - **关闭**: 保存静默模式配置并关闭窗口

### 2.4 消息区域（聊天历史）
- 容器: SScrollBox，纵向滚动，FillHeight 占满剩余空间
- 消息上限: 500 条
- 消息模型: Sender/Content/Timestamp/bIsCode
- 发送者颜色区分: user(蓝)/assistant(绿)/streaming(灰绿)/thinking(淡紫)/system(灰)
- 本地化标签

### 2.5 快捷输入栏（可折叠，位于消息列表与输入框之间）
- **位置**: 消息列表下方、工具栏上方
- **可折叠**: 使用 SExpandableArea，默认折叠，点击标题栏展开/收起
- **显示**: 折叠时显示标题标签 "快捷输入 / Quick Inputs" 和 [+ 添加] 按钮；展开后显示快捷按钮列表
- **按钮排列**: 横向 WrapBox 自动换行，每个快捷按钮显示名称
- **操作**:
  - **点击快捷按钮**: 将对应内容填入输入框并聚焦
  - **添加 (+ 添加)**: 折叠状态下标题栏右侧，弹出模态对话框，填写名称和内容后保存
  - **编辑 (e)**: 弹出模态对话框，修改名称和内容
  - **删除 (x)**: 直接删除，无需确认
- **存储**: JSON 文件持久化 (`Saved/UEAgent/quick_inputs.json`)，UTF-8 无 BOM 编码
- **加载保障**: 双重编码兼容读取 + 加载成功后自动重写确保编码一致性
- **空状态**: 无快捷输入时仅显示标题和添加按钮

### 2.6 工具栏（消息列表下方、快捷输入上方）
- **会话切换**: 下拉菜单，显示当前会话名称，点击展开会话列表（历史会话 + 删除按钮，无新建按钮）
- **+ 新会话**: 创建全新对话会话，清空当前聊天历史
- **Skills**: 打开 Skill 与 MCP 管理面板（独立 SWindow 窗口）
- **停止**: 仅在等待 AI 回复时可点击，停止当前生成，位于工具栏右侧
- **发送**: 发送消息（等待中变灰变黄），位于工具栏最右侧
- **Attach**: 打开文件选择器添加附件，也可用 Ctrl+V 粘贴剪贴板图片
- **按钮布局**: 左侧 `[会话▼] [+新会话] [Skills]`，中右 `[Attach]`，右侧 `[停止] [发送]`

### 2.7 输入区域
- 多行输入框（最小 32px，最大 120px）
- 发送模式切换: Enter / Ctrl+Enter（在状态栏展开区域配置）
- 动态占位提示文本
- / 前缀触发 Slash 命令菜单

## 三、Slash 快捷命令系统

### 3.1 命令菜单
- **触发方式**: 在输入框中输入 `/` 前缀自动弹出命令菜单
- **菜单位置**: 悬浮于输入框上方
- **实时过滤**: 根据输入内容动态过滤可用命令

### 3.2 命令列表
- **本地命令**:
  - `/connect` - 手动连接服务器
  - `/disconnect` - 断开服务器连接
  - `/diagnose` - 运行诊断检查
  - `/status` - 查看当前连接状态
  - `/clear` - 清空当前会话聊天记录
  - `/cancel` - 取消正在进行的请求
  - `/help` - 显示帮助信息
- **AI 命令**:
  - `/new` - 创建新会话
  - `/compact` - 压缩会话上下文
  - `/review` - 审阅当前操作
  - `/undo` - 撤销上一步操作

## 十二、Plan 模式

### 12.1 Plan 生成与展示
- **Plan 模式开关**: 用户可选择启用 Plan 模式，AI 会先生成执行计划而非直接执行
- **AI 生成 Plan**: AI 返回 JSON 格式的执行计划，包含步骤列表和依赖关系
- **步骤卡片**: 每个步骤以卡片形式展示，状态包括:
  - 待执行（Pending）
  - 执行中（In Progress）
  - 已完成（Completed）
  - 失败（Failed）
  - 已跳过（Skipped）

### 12.2 Plan 交互
- **审阅计划**: 用户可在执行前查看完整计划，确认步骤合理性
- **编辑计划**: 可删除尚未开始执行的步骤，调整执行顺序
- **执行计划**: 逐步骤执行，实时更新每个步骤的状态
- **暂停/继续**: 支持暂停执行流程，之后可继续从暂停处开始

### 12.3 执行控制
- **暂停全部**: 暂停所有待执行步骤
- **取消全部**: 中断整个计划执行
- **步骤失败处理**: 单步骤失败时提示用户，可选择重试、跳过或终止

## 十三、Skill 与 MCP 管理面板

### 13.1 Skill 管理
- **分层展示**: 按来源分类显示
  - official: 官方内置 Skill
  - marketplace: 社区共享 Skill
  - user: 用户自定义 Skill
  - platform: 其他平台 Skill（不属于以上三类的统一归此）
- **操作**: 启用/禁用/钉选常用 Skill
- **筛选**:
  - 层级筛选行: 全部/官方/市集/用户/其他平台 + 未安装切换按钮
  - 软件分类筛选行: 全部/UE/Maya/Max/通用
  - 搜索框: 名称/描述模糊匹配
- **列表行**: DisplayName + Version + Author（同行）, 第二行 Skill ID
- **详情弹窗**:
  - 安装路径 + 源码路径（分别显示，路径为空时标 "-"）
  - 版本不同时显示: `v0.3.0 (源码: v0.4.0)`
  - 两个"打开目录"按钮: 打开安装目录 / 打开源码目录
- **去重**: Skill 名称别名映射（旧名→新名），frontmatter name 与目录名不一致时以目录名为准

### 13.2 Skill 生命周期
- **创建**: 通过 AI 对话式引导创建新 Skill
- **编辑**: 修改现有 Skill 的配置和代码
- **发布**: 将 Skill 发布到 marketplace 或 official 频道
- **更新**: 检查并应用 Skill 更新
- **卸载/删除**: 移除不再需要的 Skill

### 13.3 MCP 管理
- **服务器状态**: 查看各 MCP 服务器的连接状态和运行情况
- **工具列表**: 浏览当前可用的 MCP 工具，了解工具功能和参数

### 13.4 面板入口
- **主入口**: Dashboard 工具栏 [Skills] 按钮，点击打开独立 SWindow 窗口（已存在则激活）
- **备用入口**: Dashboard 状态栏 [设置] → [Skills 管理]，同样打开管理面板窗口
- **窗口类型**: 独立 SWindow（非 Nomad Tab），Dashboard 销毁时自动关闭

## 十五、Agent 切换

### 15.1 Agent 选择
- **入口**: 设置面板中的 Agent 列表
- **列表刷新**: 从 Gateway RPC 拉取可用 Agent 列表（id/name/emoji）
- **切换行为**: 切换 Agent 后自动保存当前会话、切换 session key 前缀、重建会话菜单
- **会话缓存**: AgentSessionCache 按 Agent ID 缓存/恢复会话列表，切换时不丢失历史会话

### 15.2 会话自动恢复
- **UE 重启恢复**: 启动时从 Saved/UEAgent/last_session.json 读取上次 Agent ID + Session Key，自动恢复
- **实时保存**: 每次会话切换、新建、消息发送后自动保存状态，不依赖析构
- **宽限期**: Connect 成功后短暂宽限期内忽略轮询的 connected=false，避免抖动

### 16.1 上下文长度百分比显示

## 十七、附件功能（剪贴板图片/文件）

### 17.1 交互方式
- **Ctrl+V 粘贴**: 检测剪贴板内容类型
  - 图片数据（截图）→ 保存为 PNG 到临时目录 → 显示预览
  - 文件路径 → 添加为附件（图片复制到临时目录，非图片仅记录路径）
  - 普通文本 → 正常粘贴到输入框（不拦截）
- **Attach 按钮**: 工具栏中的文件选择器入口，支持多选
- **附件预览栏**: 工具栏与输入框之间，默认隐藏
  - 每个附件显示: [IMG]/[FILE] 标识 + 文件名 + 文件大小 + 删除按钮
  - 横向滚动布局（SScrollBox, Orient_Horizontal）
  - 有附件时自动显示，清空后自动隐藏

### 17.2 发送流程
- 发送时自动在消息前缀注入附件路径标签:
  `
  [Attachments]
  - Image: D:/path/to/clipboard_20260401.png
  - File: D:/path/to/readme.txt
  [/Attachments]

  用户输入的消息...
  `
- AI 通过 
ead 工具读取附件文件内容
- 发送后自动清空附件预览栏
- Slash 命令和新会话操作会清空待发送附件

### 17.3 技术细节
- **剪贴板图片读取**: Windows API (CF_DIBV5/CF_DIB) → IImageWrapper 编码 PNG
- **临时存储**: `Saved/UEAgent/attachments/` 目录
- **大小限制**: 图片文件最大 10MB
- **支持图片格式**: PNG, JPG, JPEG, BMP, GIF, WebP, TGA
