## 7. 问题排查指南

### 7.1 插件无法加载
**症状**：编辑器启动时报错，插件未加载
**排查步骤**：
1. [ ] 检查`.Build.cs`模块依赖是否正确
2. [ ] 检查`.uplugin`文件格式是否正确（JSON）
3. [ ] 查看Output Log完整错误信息
4. [ ] 确认PythonScriptPlugin已启用

### 7.2 子系统无法访问
**症状**：Python中获取子系统返回None
**排查步骤**：
1. [ ] 确认模块已正确编译
2. [ ] 检查类名是否正确（`UEAgentSubsystem`）
3. [ ] 确认`UCLASS(BlueprintType)`标记
4. [ ] 重启编辑器

### 7.3 UI不显示
**症状**：工具栏无图标，或点击无反应
**排查步骤**：
1. [ ] 检查`FUEClawBridgeStyle`是否初始化
2. [ ] 检查图标文件是否存在
3. [ ] 确认`UToolMenus::RegisterStartupCallback`已调用
4. [ ] 查看Output Log有无Slate相关错误

### 7.4 日志不显示
**症状**：Python的print不输出到Output Log
**排查步骤**：
1. [ ] 确认`init_unreal.py`已执行
2. [ ] 检查`sys.stdout`是否被重定向
3. [ ] 查看`LogUEAgent`类别是否启用
4. [ ] 确认UELogger类已定义

### 7.5 依赖安装失败
**症状**：依赖无法自动安装
**排查步骤**：
1. [ ] 检查网络连接
2. [ ] 查看pip错误信息
3. [ ] 尝试手动安装：`pip install websockets --target Lib`
4. [ ] 使用离线bundle安装
