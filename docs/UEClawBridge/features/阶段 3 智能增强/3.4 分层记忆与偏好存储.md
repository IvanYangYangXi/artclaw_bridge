## 🎯 目标与里程碑

### 核心目标
构建**分层记忆系统**（短期/中期/长期记忆）存储用户偏好、操作历史和项目特定模式，实现个性化AI体验。

### 验收标准（v2）
- [x] 短期记忆（4小时 TTL，到期压缩晋升到中期）

- [x] 长期记忆（永久，LRU+重要性淘汰）
- [x] 操作链路自动记录（成功/失败/崩溃）
- [x] 合并去重 + 文件大小控制
- [x] 旧格式自动迁移
- [x] Memory Briefing 自动注入 send_chat
- [x] 定时维护（30min 短期扫描 + 24h 完整维护）
- [x] 启动完整性检查 + 自动修复缺失共享模块
