# 1.6 快捷上下文注入(Context Injection)

**阶段**：阶段1 - 核心基础设施构建  
**重要性**：⭐⭐⭐⭐ (P1)  
**依赖**：1.5 主线程调度器(GameThread Marshalling)  
**预估工时**：1天

---

## 🎯 目标与里程碑

### 核心目标
构建**上下文注入系统**，自动收集当前编辑器状态（选中对象、当前关卡、项目设置等），为AI提供完整的执行上下文。

### 验收标准
- [ ] 自动收集当前选中Actor列表
- [ ] 获取当前编辑的资产路径
- [ ] 识别编辑器模式（关卡编辑/蓝图/材质）
- [ ] 上下文收集延迟<10ms
- [ ] 支持自定义上下文扩展
