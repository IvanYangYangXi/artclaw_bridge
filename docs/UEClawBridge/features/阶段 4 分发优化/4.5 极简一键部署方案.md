# 4.5 极简一键部署方案

**阶段**：阶段4 - 分发优化与生态  
**重要性**：⭐⭐⭐⭐ (P1)  
**依赖**：无（独立）  
**预估工时**：1-2天  
**状态**：✅ 已完成

---

## 🎯 目标与里程碑

### 核心目标
插件复制到 UE 项目后，**首次启动自动完成所有依赖安装**，无需独立安装脚本或手动操作。

### 验收标准
- [x] 插件启动时自动检测缺失依赖
- [x] 后台线程自动 `pip install --target` 定向安装，不阻塞编辑器
- [x] 支持离线 bundle 降级安装
- [x] 安装完成后自动启动 MCP 服务

### 实际实现
通过 `init_unreal.py` 内置的依赖管理流程实现：
- `_check_dependencies_fast()` — 快速同步检测已安装包
- `_install_dependencies()` — 按需安装缺失包（先尝试离线 bundle，再 pip install）
- 后台线程异步执行，不阻塞编辑器启动
- 安装完成后自动回调主线程启动 MCP Server

### 与原设计的差异
原设计为独立 `setup.bat` 脚本（5步自动化），实际采用**启动时自检**方案，用户体验更无感——无需运行任何脚本，插件目录复制到项目即可工作。
