## ⚠️ 常见问题预判

### 问题1：编译失败 - 缺少依赖
**现象**: 无法找到Python的websockets库  
**解决**: 
- 检查 `Plugins\UEEditorAgent\Content\Python\Lib\websockets` 是否存在
- 如果不存在，需要运行依赖安装脚本
- 或手动安装: `pip install --target="插件路径\Lib" websockets`

### 问题2：MCP服务器无法启动
**现象**: Output Log中没有"[MCPServer] WebSocket服务器已启动"日志  
**解决**:
- 检查端口是否被占用（使用`netstat -ano | findstr :8080`）
- 检查防火墙设置
- 查看完整的Python错误日志

### 问题3：Dashboard状态不更新
**现象**: 点击Test Connection按钮，状态不变  
**解决**:
- 检查Subsystem委托绑定是否正确
- 在Python控制台手动测试: `subsystem.set_connection_status(True)`
- 检查Output Log中的Python错误

### 问题4：工具栏按钮不显示
**现象**: 启动编辑器后找不到Agent图标按钮  
**解决**:
- 检查插件是否已启用
- 检查Output Log是否有Slate相关错误
- 尝试重启编辑器
