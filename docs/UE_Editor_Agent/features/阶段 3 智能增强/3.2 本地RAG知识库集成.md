## 🎯 目标与里程碑

### 核心目标
构建**本地RAG（Retrieval-Augmented Generation）知识库**，基于项目文档、代码、资产元数据构建向量索引，为AI提供项目特定的知识检索能力。

### 验收标准
- [ ] 支持多种文档格式（.md, .txt, .py, .cpp）
- [ ] 向量索引构建时间<5分钟（中型项目）
- [ ] 检索响应时间<500ms
- [ ] Top-5检索准确率>80%
