## 🎯 目标与里程碑

### 核心目标
构建**视口同步系统**，在AI操作时在UE视口中高亮显示受影响的对象，提供**视觉反馈**。

### 验收标准
- [ ] 选中Actor高亮显示（黄色边框）
- [ ] 操作前预览高亮（闪烁3次）
- [ ] 支持多个Actor同时高亮
- [ ] 高亮延迟<50ms
- [ ] 不影响编辑器性能

## 🔧 核心实现

```python
import unreal
from typing import List, Optional
import time

class ViewportHighlighter:
    """视口高亮器"""
    
    def __init__(self):
        self.highlighted_actors: List[unreal.Actor] = []
        self.original_colors: Dict[str, unreal.Color] = {}
    
    def highlight_actors(self, actors: List[unreal.Actor], 
                        color: unreal.Color = unreal.Color(255, 255, 0, 255)):
        """高亮Actor"""
        self.clear_highlight()
        
        for actor in actors:
            # 保存原始材质
            self._save_original_color(actor)
            
            # 应用高亮材质
            self._apply_highlight_material(actor, color)
            
            self.highlighted_actors.append(actor)
        
        unreal.log(f"[Viewport] 高亮 {len(actors)} 个Actor")
    
    def preview_highlight(self, actors: List[unreal.Actor]):
        """预览高亮（闪烁3次）"""
        def _flash():
            for i in range(3):
                self.highlight_actors(actors, unreal.Color(255, 0, 0, 255))
                time.sleep(0.2)
                self.clear_highlight()
                time.sleep(0.2)
        
        # 异步执行闪烁
        import threading
        threading.Thread(target=_flash, daemon=True).start()
    
    def clear_highlight(self):
        """清除高亮"""
        for actor in self.highlighted_actors:
            self._restore_original_color(actor)
        
        self.highlighted_actors.clear()
        self.original_colors.clear()
    
    def _save_original_color(self, actor: unreal.Actor):
        """保存原始颜色"""
        # 简化的颜色保存
        self.original_colors[actor.get_name()] = unreal.Color(255, 255, 255, 255)
    
    def _apply_highlight_material(self, actor: unreal.Actor, color: unreal.Color):
        """应用高亮材质"""
        # 实际应创建高亮材质实例
        actor.set_folder_path("_Highlighted")
    
    def _restore_original_color(self, actor: unreal.Actor):
        """恢复原始颜色"""
        actor.set_folder_path("")


# 全局实例
highlighter = ViewportHighlighter()

# 使用示例
if __name__ == "__main__":
    # 获取选中Actor
    selected = unreal.EditorLevelLibrary.get_selected_level_actors()
    
    # 预览高亮
    highlighter.preview_highlight(selected)
