# 阶段 2.1：编辑器内对话面板 (In-Editor Chat Tab)

**目标**：在UE编辑器中嵌入对话面板，提供与AI的交互界面，支持消息历史、代码高亮、实时响应，并与UE编辑器主题完美融合。

**🚩 里程碑**：
1. 创建可停靠的Chat Tab窗口
2. 嵌入SWebBrowser或原生Slate实现消息展示
3. 支持文本输入、消息发送、响应显示
4. 与UE编辑器主题（暗色/亮色）同步
5. 支持消息历史记录和滚动

**适用范围**：☑ UE_Editor_Agent ☐ Maya_Editor_Agent ☐ 其他

---

**验证清单**：
- [ ] Chat Tab可停靠到编辑器任意位置
- [ ] 支持文本输入和发送
- [ ] AI响应正确显示
- [ ] 消息可滚动查看历史
- [ ] 代码块高亮显示
- [ ] 主题与UE编辑器一致

---

**跨项目差异**：
- **UE**：使用Slate/SWebBrowser
- **Maya**：使用Qt/PySide
- **Blender**：使用Blender UI
