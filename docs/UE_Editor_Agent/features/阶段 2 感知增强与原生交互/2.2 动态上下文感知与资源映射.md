# 2.2 动态上下文感知与资源映射

**阶段**：阶段2 - 感知增强与原生交互  
**重要性**：⭐⭐⭐⭐⭐ (P0)  
**依赖**：1.6 快捷上下文注入  
**预估工时**：2-3天

---

## 🎯 目标与里程碑

### 核心目标
构建**动态上下文感知系统**，实时映射UE资源（Actor、资产、关卡）为MCP Resources，让AI能理解和操作当前编辑器状态。

### 验收标准
- [ ] 自动发现当前关卡所有Actor
- [ ] 映射选中Actor为MCP Resource
- [ ] 资源变更实时同步（延迟<100ms）
- [ ] 支持查询资源属性（位置、旋转、缩放）
- [ ] 支持资源层级关系（父子Actor）
