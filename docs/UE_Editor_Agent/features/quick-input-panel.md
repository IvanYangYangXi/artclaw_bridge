# 快捷输入分栏功能需求

## 概述
在 ArtClaw UE 插件面板中添加一个可折叠分栏，用于存放用户自定义的快捷输入文本。

## 功能需求

### 1. UI 设计
- **位置**: 插件面板中，类似"连接状态"的折叠面板样式
- **标题**: "快捷输入" 或 "Quick Inputs"
- **交互**: 可展开/折叠，节省面板空间

### 2. 快捷输入管理
- **添加**: 用户可以添加新的快捷输入项
  - 输入名称（显示用）
  - 输入内容（实际文本）
- **编辑**: 支持修改已有快捷输入
- **删除**: 支持删除不需要的快捷输入
- **排序**: 支持拖拽排序（可选）

### 3. 使用方式
- 点击快捷输入项 → 自动填充到对话框输入框
- 支持双击或单击触发（待确定）

### 4. 数据存储
- **位置**: 本地配置文件
- **路径**: 建议存储在插件配置目录
  - Windows: `%LOCALAPPDATA%/ArtClaw/quick_inputs.json`
  - 或项目目录: `D:\MyProject_D\artclaw\config\quick_inputs.json`
- **格式**: JSON
- **数量**: 不做限制

### 5. 配置示例
```json
{
  "quick_inputs": [
    {
      "id": "uuid-1",
      "name": "导出选中对象",
      "content": "帮我导出选中的对象"
    },
    {
      "id": "uuid-2",
      "name": "添加碰撞体",
      "content": "给选中对象添加碰撞体组件"
    }
  ]
}
```
