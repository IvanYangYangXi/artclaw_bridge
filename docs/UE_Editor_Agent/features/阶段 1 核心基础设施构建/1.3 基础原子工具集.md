## 核心实现代码 (atomic_tools.py)

```python
import unreal
import asyncio
from tool_manager import ToolRegistry

def run_on_main_thread(func):
    """装饰器：强制函数在 UE 主线程运行并等待结果"""
    @functools.wraps(func)
    def wrapper(*args, **kwargs):
        # 使用 Python 3.10+ 的 Future 来获取主线程执行结果
        import queue
        result_queue = queue.Queue()

        def main_thread_exec():
            try:
                res = func(*args, **kwargs)
                result_queue.put(("SUCCESS", res))
            except Exception as e:
                result_queue.put(("ERROR", str(e)))

        unreal.get_interpreter().execute_on_main_thread(main_thread_exec)
        status, value = result_queue.get(timeout=5.0) # 等待主线程返回
        if status == "SUCCESS": return value
        raise RuntimeError(value)
    return wrapper

# --- 基础工具定义 ---

@ToolRegistry.register_to_mcp(name="get_selected_actors")
@run_on_main_thread
def get_selected_actors() -> list:
    """获取当前编辑器中选中的所有 Actor 名称。"""
    selected = unreal.EditorLevelLibrary.get_selected_level_actors()
    return [a.get_name() for a in selected]

@ToolRegistry.register_to_mcp(name="spawn_basic_shape")
@run_on_main_thread
def spawn_basic_shape(shape_type: str, x: float, y: float, z: float):
    """
    在场景中生成基础形状。
    :param shape_type: 形状类型，如 'Cube', 'Sphere', 'Cylinder'
    """
    path_map = {
        "Cube": "/Engine/BasicShapes/Cube",
        "Sphere": "/Engine/BasicShapes/Sphere"
    }
    asset_path = path_map.get(shape_type, path_map["Cube"])
    location = unreal.Vector(x, y, z)
    return unreal.EditorLevelLibrary.spawn_actor_from_path(asset_path, location).get_name()
```

## 关键技术细节
1. **阻塞与非阻塞**：`run_on_main_thread` 使用了 `Queue` 进行同步等待。这确保了 MCP 响应（Response）是在操作真正完成后才发出的。
2. **错误处理**：如果主线程操作失败（如资产路径不存在），错误会被捕获并传回 WebSocket 线程，最终作为 JSON-RPC Error 返回给 OpenClaw，让 AI 知道原因。
3. **坐标系适配**：UE 使用的是厘米（cm）单位且 Z 轴向上。在 Tool 描述中需要明确告知 AI 这一点，防止生成的物体因坐标过小而看不见。

**🚩 里程碑**：
1. 在 OpenClaw 中输入“获取当前选中的物体”，能准确返回 UE 编辑器中选中的列表。
2. 输入“在坐标 (0,0,100) 生成一个方块”，UE 场景中即时出现对应的 Actor。
3. 即使在生成过程中猛烈操作编辑器界面，也不会因为多线程冲突导致 Crash。
