# 阶段 1.2：万能执行器 (Universal Proxy)

**目标**：开发核心 Tool `run_ue_python(code: str)`，通过 `exec()` 函数执行任意 Python 代码，并内置 `try-except` 捕获所有异常，将 Traceback 转换为字符串返回给 AI，实现安全且强大的代码执行能力。

**🚩 里程碑**：
1. 实现 `run_ue_python` 工具，注册到 MCP 工具列表
2. AI 可以发送任意 Python 代码并在 UE 编辑器中执行
3. 执行结果或错误信息能正确返回给 AI
4. 发生错误时不会导致编辑器崩溃
5. 高危操作（如删除大量 Actor）需要后续阶段的安全机制

**适用范围**：☑ UE_Editor_Agent ☐ Maya_Editor_Agent ☐ 其他

---

## 1. 背景与上下文

在阶段 1.1 中，我们建立了 WebSocket 通信网关。现在需要实现第一个核心 Tool——万能执行器。这是整个系统的核心功能，允许 AI 通过自然语言生成 Python 代码并在 UE 编辑器中执行。

**设计考虑**：
- **安全性**：使用 `try-except` 捕获所有异常，防止代码错误导致编辑器崩溃
- **反馈机制**：将执行结果、返回值、异常信息都返回给 AI
- **上下文注入**：自动注入常用对象（S、W、L），减少 AI 代码量
- **性能**：避免阻塞主线程，需要结合阶段 1.5 的主线程调度器

**安全警告**：
此工具具有极高权限，可以执行任意 Python 代码。必须在后续阶段（1.3-1.4）中添加静态预审和事务保护机制。

## 2. 详细实现步骤

### 2.1 创建万能执行器工具类

在 `Content/Python/` 目录下创建 `tools/` 目录，然后创建 `universal_proxy.py`。

**操作步骤**：
1. 创建 `Content/Python/tools/` 目录
2. 创建 `universal_proxy.py` 文件
3. 实现 `run_ue_python` 函数
4. 添加异常捕获和结果格式化逻辑
5. 创建工具定义（Tool Definition）供 MCP 注册

**注意事项**：
- 使用 `exec()` 执行代码，而不是 `eval()`，因为代码可能是多条语句
- 捕获 `Exception` 而不是特定异常，确保所有错误都能捕获
- 使用 `traceback` 模块获取完整的堆栈信息
- 将执行环境（globals）与主环境隔离，避免污染

### 2.2 集成到 MCP 服务器

修改 `mcp_server.py`，将 `run_ue_python` 工具注册到 MCP 工具列表。

**操作步骤**：
1. 在 `mcp_server.py` 中导入 `run_ue_python`
2. 创建工具定义字典，包含名称、描述、参数模式
3. 在 `handle_message` 中处理 `tools/call` 请求
4. 调用 `run_ue_python` 并返回结果

**注意事项**：
- 遵循 MCP 协议的工具调用格式（JSON-RPC 2.0）
- 参数验证：确保 `code` 参数存在且为字符串
- 结果格式化：区分成功和失败的情况

### 2.3 上下文注入（可选，阶段 1.6 会详细实现）

在 `exec()` 的 `globals` 参数中预置常用对象，减少 AI 代码量。

**预置对象**：
- `S`: 当前选中的 Actor 列表（`unreal.EditorLevelLibrary.get_selected_level_actors()`）
- `W`: 当前世界（`unreal.EditorLevelLibrary.get_editor_world()`）
- `L`: 常用库简写（如 `unreal`）

**注意事项**：
- 这些对象在每次执行前动态获取，确保是最新状态
- 避免覆盖用户代码中已有的同名变量

### 3.1 Python - 万能执行器工具

```python
# Content/Python/tools/universal_proxy.py
"""
万能执行器 - 执行任意Python代码并返回结果
"""

import sys
import traceback
from typing import Dict, Any, Optional, Tuple
import unreal


class UniversalProxy:
    """万能执行器，安全地执行任意Python代码"""
    
    def __init__(self):
        """初始化执行器"""
        self.execution_count = 0
        self.last_result = None
        self.last_error = None
    
    def run_python_code(self, code: str, inject_context: bool = True) -> Dict[str, Any]:
        """
        执行Python代码并返回结果
        
        Args:
            code: 要执行的Python代码字符串
            inject_context: 是否注入上下文对象（S, W, L）
            
        Returns:
            执行结果字典，包含以下字段：
            - success: bool, 是否执行成功
            - result: Any, 执行结果（如果有）
            - output: str, 标准输出内容
            - error: str, 错误信息（如果失败）
            - execution_time: float, 执行耗时（秒）
            - execution_count: int, 执行序号
        """
        import time
        
        self.execution_count += 1
        start_time = time.time()
        
        # 创建执行环境
        exec_globals = self._create_execution_environment(inject_context)
        exec_locals = {}
        
        # 重定向标准输出
        old_stdout = sys.stdout
        old_stderr = sys.stderr
        
        try:
            # 使用StringIO捕获输出
            from io import StringIO
            output_capture = StringIO()
            sys.stdout = output_capture
            sys.stderr = output_capture
            
            # 执行代码
            # NOTE: 使用exec执行代码，最后一个表达式的结果会作为返回值
            result = exec(code, exec_globals, exec_locals)
            
            # 获取输出内容
            output_content = output_capture.getvalue()
            
            # 获取执行结果
            # 如果代码中有变量 '_'，通常表示最后一个表达式的结果
            final_result = exec_locals.get('_', result)
            
            execution_time = time.time() - start_time
            
            # 记录成功结果
            self.last_result = final_result
            self.last_error = None
            
            unreal.log(f"[UniversalProxy] 代码执行成功 (耗时: {execution_time:.3f}s)")
            
            return {
                "success": True,
                "result": self._serialize_result(final_result),
                "output": output_content,
                "error": None,
                "execution_time": execution_time,
                "execution_count": self.execution_count
            }
            
        except Exception as e:
            # 捕获所有异常
            execution_time = time.time() - start_time
            
            # 获取完整的错误信息
            error_message = f"{type(e).__name__}: {str(e)}"
            error_traceback = traceback.format_exc()
            
            # 记录错误
            self.last_result = None
            self.last_error = error_message
            
            unreal.log_error(f"[UniversalProxy] 代码执行失败 (耗时: {execution_time:.3f}s)")
            unreal.log_error(error_traceback)
            
            return {
                "success": False,
                "result": None,
                "output": output_capture.getvalue() if 'output_capture' in locals() else "",
                "error": f"{error_message}\n\n{error_traceback}",
                "execution_time": execution_time,
                "execution_count": self.execution_count
            }
            
        finally:
            # 恢复标准输出
            sys.stdout = old_stdout
            sys.stderr = old_stderr
    
    def _create_execution_environment(self, inject_context: bool) -> Dict[str, Any]:
        """
        创建代码执行环境
        
        Args:
            inject_context: 是否注入上下文对象
            
        Returns:
            执行环境字典（globals）
        """
        # 基础环境
        exec_globals = {
            "__builtins__": __builtins__,
            "unreal": unreal,
        }
        
        # 注入常用上下文对象（如果启用）
        if inject_context:
            try:
                # S: 选中的Actor列表
                selected_actors = unreal.EditorLevelLibrary.get_selected_level_actors()
                exec_globals["S"] = selected_actors
                
                # W: 当前世界
                current_world = unreal.EditorLevelLibrary.get_editor_world()
                exec_globals["W"] = current_world
                
                # L: 常用库（这里可以添加更多快捷方式）
                exec_globals["L"] = {
                    "unreal": unreal,
                    # 可以添加更多常用库的快捷方式
                }
                
            except Exception as e:
                unreal.log_warning(f"[UniversalProxy] 注入上下文失败: {e}")
        
        return exec_globals
    
    def _serialize_result(self, result: Any) -> Any:
        """
        序列化执行结果，转换为可JSON序列化的格式
        
        Args:
            result: 执行结果
            
        Returns:
            序列化后的结果
        """
        if result is None:
            return None
        
        # 基本类型直接返回
        if isinstance(result, (int, float, str, bool)):
            return result
        
        # 列表和元组
        if isinstance(result, (list, tuple)):
            return [self._serialize_result(item) for item in result]
        
        # 字典
        if isinstance(result, dict):
            return {str(k): self._serialize_result(v) for k, v in result.items()}
        
        # Unreal对象
        if hasattr(result, "get_name"):
            return f"<Unreal Object: {result.get_name()}>"
        
        # 其他对象转换为字符串
        try:
            return str(result)
        except:
            return "<无法序列化的对象>"


# 全局执行器实例
_default_executor = None


def get_default_executor() -> UniversalProxy:
    """获取默认的万能执行器实例"""
    global _default_executor
    if _default_executor is None:
        _default_executor = UniversalProxy()
    return _default_executor


def run_ue_python(code: str, inject_context: bool = True) -> Dict[str, Any]:
    """
    执行Python代码的便捷函数
    
    Args:
        code: 要执行的Python代码
        inject_context: 是否注入上下文对象
        
    Returns:
        执行结果字典
        
    Example:
        result = run_ue_python("""
            import unreal
            actors = unreal.EditorLevelLibrary.get_selected_level_actors()
            print(f"选中了 {len(actors)} 个Actor")
        """)
        
        if result["success"]:
            print(f"执行成功: {result['result']}")
        else:
            print(f"执行失败: {result['error']}")
    """
    executor = get_default_executor()
    return executor.run_python_code(code, inject_context)


# MCP工具定义（供服务器注册使用）
TOOL_DEFINITION = {
    "name": "run_ue_python",
    "description": "在Unreal Editor中执行任意Python代码。可以访问unreal模块、操作场景、创建对象等。代码会在try-except中执行，错误会返回详细信息。",
    "inputSchema": {
        "type": "object",
        "properties": {
            "code": {
                "type": "string",
                "description": "要执行的Python代码字符串。可以包含多行代码、函数定义、循环等。"
            },
            "inject_context": {
                "type": "boolean",
                "description": "是否注入上下文对象（S=选中Actor, W=当前世界）。默认为True。",
                "default": True
            }
        },
        "required": ["code"]
    }
}
```

### 3.2 Python - MCP服务器集成

```python
# Content/Python/mcp_server.py (在1.1基础上添加)

# ... 前面的代码保持不变 ...

# 导入万能执行器
from tools.universal_proxy import run_ue_python, TOOL_DEFINITION as UniversalProxyTool

# 工具注册表
REGISTERED_TOOLS = {
    UniversalProxyTool["name"]: {
        "definition": UniversalProxyTool,
        "handler": run_ue_python
    }
}

class MCPServer:
    # ... __init__, find_available_port, start, stop 等方法保持不变 ...
    
    async def handle_message(self, message: str):
        """处理接收到的消息"""
        try:
            data = json.loads(message)
            message_id = data.get("id")
            method = data.get("method")
            
            unreal.log(f"[MCPServer] 收到消息: {method}")
            
            # 处理工具调用
            if method == "tools/list":
                # 返回工具列表
                response = {
                    "jsonrpc": "2.0",
                    "id": message_id,
                    "result": {
                        "tools": [tool["definition"] for tool in REGISTERED_TOOLS.values()]
                    }
                }
                
            elif method == "tools/call":
                # 调用工具
                params = data.get("params", {})
                tool_name = params.get("name")
                tool_args = params.get("arguments", {})
                
                if tool_name not in REGISTERED_TOOLS:
                    response = {
                        "jsonrpc": "2.0",
                        "id": message_id,
                        "error": {
                            "code": -32601,
                            "message": f"Tool '{tool_name}' not found"
                        }
                    }
                else:
                    # 调用工具处理器
                    try:
                        tool_info = REGISTERED_TOOLS[tool_name]
                        handler = tool_info["handler"]
                        
                        unreal.log(f"[MCPServer] 调用工具: {tool_name}")
                        unreal.log(f"[MCPServer] 参数: {tool_args}")
                        
                        # 执行工具函数
                        result = handler(**tool_args)
                        
                        response = {
                            "jsonrpc": "2.0",
                            "id": message_id,
                            "result": result
                        }
                        
                    except Exception as e:
                        unreal.log_error(f"[MCPServer] 工具调用失败: {e}")
                        response = {
                            "jsonrpc": "2.0",
                            "id": message_id,
                            "error": {
                                "code": -32603,
                                "message": f"Tool execution failed: {str(e)}"
                            }
                        }
            
            else:
                # 未知方法
                response = {
                    "jsonrpc": "2.0",
                    "id": message_id,
                    "error": {
                        "code": -32601,
                        "message": f"Method '{method}' not found"
                    }
                }
            
            # 发送响应
            await self.send_response(response)
            
        except Exception as e:
            unreal.log_error(f"[MCPServer] 处理消息失败: {e}")
            error_response = {
                "jsonrpc": "2.0",
                "id": None,
                "error": {
                    "code": -32603,
                    "message": "Internal error",
                    "data": str(e)
                }
            }
            await self.send_response(error_response)
```

## 6. 常见问题与解决方案

### 问题1：exec() 返回值总是 None

**现象**：
```python
result = run_ue_python("1 + 1")
print(result["result"])  # 输出 None
```

**原因**：
`exec()` 执行表达式不会返回值，只会执行语句

**解决方案**：
1. 在代码中将结果赋值给变量：
   ```python
   code = """
   result = 1 + 1
   result  # 最后一行是变量，会作为返回值
   """
   ```

2. 或者使用 `eval()` 执行单个表达式：
   ```python
   # 在 run_python_code 中特殊处理
   if '\n' not in code.strip():
       result = eval(code, exec_globals, exec_locals)
   else:
       result = exec(code, exec_globals, exec_locals)
   ```

### 问题2：Unreal 对象无法序列化

**现象**：
```
TypeError: Object of type Actor is not JSON serializable
```

**原因**：
Unreal 的 UObject 无法直接转换为 JSON

**解决方案**：
在 `_serialize_result` 中处理 Unreal 对象：
```python
if hasattr(result, "get_name"):
    return f"<Unreal Object: {result.get_name()}>"
```

或者只返回关键属性：
```python
if isinstance(result, unreal.Actor):
    return {
        "name": result.get_name(),
        "class": result.get_class().get_name(),
        "location": str(result.get_actor_location())
    }
```

### 问题3：标准输出捕获不完整

**现象**：
某些 print 输出没有捕获到

**原因**：
Unreal 的 log 系统可能绕过了 Python 的 stdout

**解决方案**：
1. 重定向 unreal.log：
   ```python
   # 在 init_unreal.py 中
   import unreal
   import sys
   from io import StringIO
   
   class UELogRedirect:
       def __init__(self):
           self.buffer = StringIO()
       
       def log(self, message):
           self.buffer.write(message + "\n")
   
   redirect = UELogRedirect()
   unreal.log = redirect.log
   ```

2. 或者同时使用 `unreal.log` 和 stdout 重定向

### 问题4：执行耗时过长导致 UI 卡顿

**现象**：
执行复杂代码时，编辑器界面卡顿

**原因**：
代码在主线程执行，阻塞了 UI 渲染

**解决方案**：
1. **临时方案**：将代码拆分成小块执行
2. **最终方案**：等待阶段 1.5 实现主线程调度器：
   ```python
   # 在阶段 1.5 中实现
   def run_on_game_thread(func):
       unreal.get_interpreter().execute_on_main_thread(func)
   ```

### 问题5：上下文对象 S/W/L 不存在

**现象**：
```
NameError: name 'S' is not defined
```

**原因**：
上下文注入未生效或注入失败

**解决方案**：
1. 确保调用时 `inject_context=True`
2. 检查 `universal_proxy.py` 中的 `_create_execution_environment` 方法
3. 在代码中手动获取选中对象：
   ```python
   import unreal
   S = unreal.EditorLevelLibrary.get_selected_level_actors()
   ```

### 问题6：MCP 客户端收不到响应

**现象**：
AI 发送工具调用请求后，没有收到响应

**原因**：
1. WebSocket 连接已断开
2. 服务器未正确处理 `tools/call` 请求
3. 响应格式不符合 MCP 协议

**解决方案**：
1. 检查 WebSocket 连接状态
2. 确认 `mcp_server.py` 中的 `handle_message` 能处理 `tools/call`
3. 验证响应格式：
   ```python
   # 必须有 jsonrpc, id, result/error 字段
   response = {
       "jsonrpc": "2.0",
       "id": message_id,  # 必须与请求中的 id 相同
       "result": {...}    # 或 "error": {...}
   }
   ```
