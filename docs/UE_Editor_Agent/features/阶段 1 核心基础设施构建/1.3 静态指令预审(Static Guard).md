## 🎯 目标与里程碑

### 核心目标
构建**静态指令预审系统**，在代码执行前通过AST分析拦截危险操作，实现**零运行时风险**的安全沙箱。

### 验收标准
- [ ] 100%拦截`os.system`, `subprocess`等危险调用
- [ ] 敏感文件访问（如`C:\Windows\System32`）被阻止
- [ ] 网络访问必须白名单授权
- [ ] 误报率<5%（合法代码被错误拦截）
- [ ] 预审延迟<50ms（1000行代码）

---

## 📚 背景与上下文

### 为什么需要Static Guard？

在**万能执行器**中，我们允许AI生成的Python代码在UE环境中运行。但如果直接执行未经检查的代码，可能导致：

```python
# ❌ 危险示例：AI可能生成的恶意/误操作代码
import os
os.system("del /f /s /q C:\\*.*")  # 灾难性后果！

import subprocess
subprocess.run(["format", "D:"])  # 格式化硬盘！

# 或者误操作：
import shutil
shutil.rmtree("C:/MyProject/")  # 删除整个项目！
```

### 设计原则

1. **静态优于动态**：在AST层面拦截，不执行代码
2. **白名单优先**：默认拒绝，显式允许
3. **可配置**：支持项目级安全策略
4. **可审计**：记录所有拦截日志

## 🔧 详细实现步骤

### 步骤1：AST扫描器核心实现

创建`StaticGuard/ast_scanner.py`：

```python
import ast
import logging
from typing import Set, List, Dict, Any, Optional
from enum import Enum

class RiskLevel(Enum):
    SAFE = "safe"          # 完全安全
    LOW = "low"            # 低风险（如文件读取）
    MEDIUM = "medium"      # 中风险（如文件写入）
    HIGH = "high"          # 高风险（如系统命令）
    CRITICAL = "critical"  # 致命风险（格式化、删除系统文件）

class SecurityIssue:
    def __init__(self, risk_level: RiskLevel, message: str, 
                 line: int, col: int, node: ast.AST):
        self.risk_level = risk_level
        self.message = message
        self.line = line
        self.col = col
        self.node = node
        
    def to_dict(self) -> Dict[str, Any]:
        return {
            "level": self.risk_level.value,
            "message": self.message,
            "line": self.line,
            "col": self.col,
            "code": ast.unparse(self.node) if hasattr(ast, 'unparse') else str(self.node)
        }

class ASTScanner(ast.NodeVisitor):
    """
    AST安全扫描器
    通过访问AST节点检测潜在风险
    """
    
    # 危险函数黑名单
    DANGEROUS_CALLS = {
        # 系统命令
        'os.system', 'os.popen', 'os.popen2', 'os.popen3', 'os.popen4',
        'subprocess.call', 'subprocess.check_call', 'subprocess.check_output',
        'subprocess.run', 'subprocess.Popen',
        # 文件系统
        'shutil.rmtree', 'shutil.move',  # move可能覆盖文件
        # 网络（需要白名单）
        'urllib.request.urlopen', 'requests.get', 'requests.post',
        # UE特定（需要授权）
        'unreal.EditorAssetLibrary.delete_asset',
        'unreal.EditorLevelLibrary.destroy_actor',
    }
    
    # 敏感路径
    SENSITIVE_PATHS = {
        'C:/Windows', 'C:/Program Files', 'C:/ProgramData',
        '/usr', '/etc', '/bin', '/sbin', '/System',
    }
    
    def __init__(self, allowed_imports: Optional[Set[str]] = None,
                 allowed_calls: Optional[Set[str]] = None):
        self.issues: List[SecurityIssue] = []
        self.allowed_imports = allowed_imports or set()
        self.allowed_calls = allowed_calls or set()
        self.current_function: Optional[str] = None
        
    def scan(self, code: str, filename: str = "<unknown>") -> List[SecurityIssue]:
        """扫描代码并返回所有安全问题"""
        try:
            tree = ast.parse(code, filename=filename)
            self.visit(tree)
        except SyntaxError as e:
            self.issues.append(SecurityIssue(
                RiskLevel.HIGH,
                f"语法错误: {e.msg}",
                e.lineno or 1,
                e.offset or 0,
                ast.Pass()  # 占位节点
            ))
        return self.issues
    
    def add_issue(self, level: RiskLevel, message: str, node: ast.AST):
        """添加安全问题"""
        self.issues.append(SecurityIssue(
            level, message, node.lineno or 1, node.col_offset or 0, node
        ))
    
    # ========================================================================
    # 危险调用检测
    # ========================================================================
    
    def visit_Call(self, node: ast.Call):
        """检查函数调用"""
        call_name = self.get_call_name(node.func)
        
        if call_name in self.DANGEROUS_CALLS:
            if call_name not in self.allowed_calls:
                self.add_issue(RiskLevel.CRITICAL, f"禁止使用危险函数: {call_name}", node)
        
        # 检查shutil.rmtree的参数
        if call_name == 'shutil.rmtree':
            self._check_rmtree_path(node)
        
        # 检查os.system的参数
        if call_name in ('os.system', 'os.popen'):
            self._check_system_command(node)
            
        # 检查文件操作
        if call_name in ('open', 'builtins.open'):
            self._check_file_open(node)
            
        self.generic_visit(node)
    
    def get_call_name(self, node: ast.AST) -> Optional[str]:
        """获取调用的完整名称，如os.system"""
        parts = []
        current = node
        
        while isinstance(current, ast.Attribute):
            parts.insert(0, current.attr)
            current = current.value
        
        if isinstance(current, ast.Name):
            parts.insert(0, current.id)
        
        return '.'.join(parts) if parts else None
    
    def _check_rmtree_path(self, node: ast.Call):
        """检查rmtree的路径参数"""
        if node.args:
            path_arg = node.args[0]
            if isinstance(path_arg, ast.Constant) and isinstance(path_arg.value, str):
                path = path_arg.value
                if self._is_sensitive_path(path):
                    self.add_issue(RiskLevel.CRITICAL, 
                                   f"禁止删除敏感路径: {path}", node)
    
    def _check_system_command(self, node: ast.Call):
        """检查系统命令"""
        if node.args:
            cmd_arg = node.args[0]
            if isinstance(cmd_arg, ast.Constant) and isinstance(cmd_arg.value, str):
                cmd = cmd_arg.value
                dangerous_cmds = ['del', 'rm', 'format', 'mkfs', 'dd']
                if any(dcmd in cmd.lower() for dcmd in dangerous_cmds):
                    self.add_issue(RiskLevel.CRITICAL, 
                                   f"危险系统命令: {cmd}", node)
    
    def _check_file_open(self, node: ast.Call):
        """检查文件打开操作"""
        if len(node.args) >= 2:
            mode_arg = node.args[1]
            if isinstance(mode_arg, ast.Constant) and isinstance(mode_arg.value, str):
                mode = mode_arg.value
                if 'w' in mode or 'a' in mode or '+' in mode:
                    # 写模式
                    if len(node.args) >= 1:
                        path_arg = node.args[0]
                        if isinstance(path_arg, ast.Constant) and isinstance(path_arg.value, str):
                            path = path_arg.value
                            if self._is_sensitive_path(path):
                                self.add_issue(RiskLevel.HIGH,
                                               f"写入敏感路径: {path}", node)
    
    def _is_sensitive_path(self, path: str) -> bool:
        """判断路径是否敏感"""
        path_lower = path.lower()
        return any(sensitive.lower() in path_lower for sensitive in self.SENSITIVE_PATHS)
    
    # ========================================================================
    # 危险import检测
    # ========================================================================
    
    def visit_Import(self, node: ast.Import):
        """检查import语句"""
        for alias in node.names:
            if alias.name in ('os', 'subprocess', 'shutil', 'urllib'):
                if alias.name not in self.allowed_imports:
                    self.add_issue(RiskLevel.MEDIUM, 
                                   f"谨慎使用模块: {alias.name}", node)
        self.generic_visit(node)
    
    def visit_ImportFrom(self, node: ast.ImportFrom):
        """检查from import语句"""
        if node.module:
            if node.module in ('os', 'subprocess', 'shutil'):
                if node.module not in self.allowed_imports:
                    self.add_issue(RiskLevel.MEDIUM, 
                                   f"谨慎使用模块: {node.module}", node)
        self.generic_visit(node)
    
    # ========================================================================
    # 危险操作检测
    # ========================================================================
    
    def visit_Delete(self, node: ast.Delete):
        """检查del语句"""
        self.add_issue(RiskLevel.LOW, "删除操作需小心", node)
        self.generic_visit(node)


class SecurityPolicy:
    """安全策略配置"""
    
    def __init__(self, config_path: Optional[str] = None):
        self.allowed_imports = set()
        self.allowed_calls = set()
        self.max_file_size = 1024 * 1024  # 1MB
        self.load_config(config_path)
    
    def load_config(self, config_path: Optional[str]):
        """从配置文件加载安全策略"""
        import json
        
        default_config = {
            "allowed_imports": ["os", "shutil"],  # 允许但需要检查
            "allowed_calls": [
                "shutil.copy",
                "os.path.join",
                "os.path.exists"
            ],
            "max_file_size": 1048576
        }
        
        if config_path and os.path.exists(config_path):
            try:
                with open(config_path, 'r', encoding='utf-8') as f:
                    config = json.load(f)
                    self.allowed_imports = set(config.get("allowed_imports", []))
                    self.allowed_calls = set(config.get("allowed_calls", []))
                    self.max_file_size = config.get("max_file_size", 1048576)
                    return
            except Exception as e:
                logging.warning(f"加载安全策略失败: {e}, 使用默认配置")
        
        # 使用默认配置
        self.allowed_imports = set(default_config["allowed_imports"])
        self.allowed_calls = set(default_config["allowed_calls"])
        self.max_file_size = default_config["max_file_size"]


class StaticGuard:
    """
    静态指令预审主类
    集成AST扫描和安全策略
    """
    
    def __init__(self, policy: Optional[SecurityPolicy] = None):
        self.policy = policy or SecurityPolicy()
        
    def check_code(self, code: str, filename: str = "<unknown>") -> Dict[str, Any]:
        """
        检查代码安全性
        
        Args:
            code: 要检查的代码字符串
            filename: 文件名（用于错误报告）
            
        Returns:
            {
                "safe": bool,  # 是否安全
                "issues": List[Dict],  # 安全问题列表
                "risk_level": str,  # 最高风险等级
                "blocked": bool  # 是否应阻止执行
            }
        """
        # 检查文件大小
        if len(code.encode('utf-8')) > self.policy.max_file_size:
            return {
                "safe": False,
                "issues": [{
                    "level": "high",
                    "message": f"代码文件过大: {len(code)} 字节"
                }],
                "risk_level": "high",
                "blocked": True
            }
        
        # AST扫描
        scanner = ASTScanner(
            allowed_imports=self.policy.allowed_imports,
            allowed_calls=self.policy.allowed_calls
        )
        issues = scanner.scan(code, filename)
        
        # 分类问题
        critical_issues = [i for i in issues if i.risk_level == RiskLevel.CRITICAL]
        high_issues = [i for i in issues if i.risk_level == RiskLevel.HIGH]
        
        # 决策逻辑
        blocked = len(critical_issues) > 0
        safe = len(issues) == 0
        
        # 确定最高风险等级
        if critical_issues:
            max_risk = RiskLevel.CRITICAL
        elif high_issues:
            max_risk = RiskLevel.HIGH
        elif issues:
            max_risk = max(i.risk_level for i in issues)
        else:
            max_risk = RiskLevel.SAFE
        
        return {
            "safe": safe,
            "issues": [i.to_dict() for i in issues],
            "risk_level": max_risk.value,
            "blocked": blocked
        }


# 使用示例
if __name__ == "__main__":
    # 示例代码
    test_codes = [
        # 安全代码
        '''
import os
import unreal

# 安全操作
project_dir = os.path.join(unreal.Paths.project_dir(), "Content")
print(f"项目目录: {project_dir}")
        ''',
        
        # 危险代码
        '''

import shutil

# 危险操作！

shutil.rmtree("C:/MyImportantProject/")
        ''',
        
        # 需要审查的代码
        '''

import json

response = requests.get("https://api.openai.com/v1/models")

        '''
    ]
    
    guard = StaticGuard()
    
    for i, code in enumerate(test_codes):
        print(f"\n{'='*60}")
        print(f"测试用例 {i+1}:")
        print(f"{'='*60}")
        result = guard.check_code(code)
        print(f"安全: {result['safe']}")
        print(f"风险等级: {result['risk_level']}")
        print(f"阻止执行: {result['blocked']}")
        if result['issues']:
            print(f"发现 {len(result['issues'])} 个问题:")
            for issue in result['issues']:
                print(f"  [{issue['level']}] 第{issue['line']}行: {issue['message']}")
```

## ❓ 常见问题

### Q1: 如何允许特定的"危险"操作？
A: 在`security_policy.json`中添加`allowed_calls`：
```json
"allowed_calls": ["shutil.rmtree"]
```
或编程方式：
```python
policy = SecurityPolicy()
policy.allowed_calls.add("shutil.rmtree")
```

### Q2: 如何添加自定义的危险模式？
A: 继承`ASTScanner`并重写`visit_Call`：
```python
class CustomScanner(ASTScanner):
    def visit_Call(self, node):
        # 检查自定义危险模式
        if self.get_call_name(node) == "my_dangerous_func":
            self.add_issue(RiskLevel.HIGH, "自定义危险函数", node)
        super().visit_Call(node)
```

### Q3: 性能太慢怎么办？
A: 
- 缓存AST扫描结果（代码哈希）
- 增量扫描（只扫描变更部分）
- 并行扫描（多文件场景）
