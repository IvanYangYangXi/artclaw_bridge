# 1.4 事务保护(Undo Guard)

**阶段**：阶段1 - 核心基础设施构建  
**重要性**：⭐⭐⭐⭐⭐ (P0)  
**依赖**：1.3 静态指令预审(Static Guard)  
**预估工时**：2-3天

---

## 🎯 目标与里程碑

### 核心目标
构建**事务保护系统**，将所有AI生成的操作包装在可撤销的事务中，实现**操作可回滚**，防止AI误操作导致的数据丢失。

### 验收标准
- [ ] 100% AI操作可被撤销（Ctrl+Z）
- [ ] 支持事务嵌套（最多5层）
- [ ] 批量操作原子性（全部成功或全部回滚）
- [ ] 崩溃恢复后仍可撤销（编辑器未关闭）
- [ ] 性能开销<10%（与直接操作对比）

---

## 📚 背景与上下文

### 为什么需要Undo Guard？

即使通过了Static Guard的代码，AI仍可能产生**逻辑错误**：

```python
# AI生成的代码可能逻辑错误
import unreal

# 错误：删除了所有静态网格体，而不是特定的那个
all_actors = unreal.EditorLevelLibrary.get_all_level_actors()
for actor in all_actors:
    if isinstance(actor, unreal.StaticMeshActor):
        actor.destroy_actor()  # 😱 删除了所有静态网格体！
```

没有Undo Guard的后果：
- 开发者只能手动重做所有工作
- 可能导致数小时工作丢失
- 降低开发者对AI助手的信任

### 设计原则

1. **透明化**：对AI代码无侵入，自动包装
2. **完整性**：捕获所有副作用（文件、资产、场景）
3. **可嵌套**：支持复杂操作的事务组合
4. **可审计**：记录所有操作日志

## 🔧 详细实现步骤

### 步骤1：UE事务管理器封装

创建`UndoGuard/undo_manager.py`：

```python
import unreal
import threading
from typing import List, Callable, Any, Optional, Dict
from contextlib import contextmanager
import time
import json

class UndoTransaction:
    """单个事务包装器"""
    
    def __init__(self, name: str, context: str = "AI操作"):
        self.name = name
        self.context = context
        self.start_time = time.time()
        self.end_time: Optional[float] = None
        self.success: bool = False
        self.error: Optional[str] = None
        self.snapshot: Optional[Dict[str, Any]] = None
        
    def to_dict(self) -> Dict[str, Any]:
        return {
            "name": self.name,
            "context": self.context,
            "start_time": self.start_time,
            "end_time": self.end_time,
            "duration": (self.end_time or time.time()) - self.start_time,
            "success": self.success,
            "error": self.error
        }


class UndoGuard:
    """
    事务保护系统
    自动包装AI操作为可撤销事务
    """
    
    def __init__(self):
        self.transaction_stack: List[UndoTransaction] = []
        self.max_stack_depth = 5
        self.enabled = True
        
    def begin_transaction(self, name: str, context: str = "AI操作") -> UndoTransaction:
        """
        开始一个新事务
        
        Args:
            name: 事务名称（显示在UE撤销栈中）
            context: 操作上下文
            
        Returns:
            UndoTransaction对象
        """
        if not self.enabled:
            return UndoTransaction(name, context)
            
        # 检查栈深度
        if len(self.transaction_stack) >= self.max_stack_depth:
            raise RuntimeError(f"事务嵌套层数超过上限({self.max_stack_depth})")
        
        # 开始UE事务
        unreal.SystemLibrary.begin_transaction(name, context)
        
        # 创建事务对象
        tx = UndoTransaction(name, context)
        self.transaction_stack.append(tx)
        
        unreal.log(f"[UndoGuard] 开始事务: {name} (深度: {len(self.transaction_stack)})")
        return tx
    
    def end_transaction(self, success: bool = True):
        """
        结束当前事务
        
        Args:
            success: 事务是否成功
        """
        if not self.enabled or not self.transaction_stack:
            return
        
        tx = self.transaction_stack[-1]
        tx.success = success
        tx.end_time = time.time()
        
        # 结束UE事务
        unreal.SystemLibrary.end_transaction()
        
        if success:
            unreal.log(f"[UndoGuard] 事务完成: {tx.name} ({tx.duration:.2f}s)")
        else:
            unreal.log_warning(f"[UndoGuard] 事务失败: {tx.name} - 已回滚")
        
        # 从事务栈移除
        self.transaction_stack.pop()
    
    def cancel_transaction(self):
        """取消当前事务并回滚"""
        if not self.enabled or not self.transaction_stack:
            return
            
        tx = self.transaction_stack[-1]
        tx.success = False
        tx.end_time = time.time()
        
        # 回滚UE事务
        unreal.SystemLibrary.cancel_transaction()
        
        unreal.log_error(f"[UndoGuard] 事务取消: {tx.name}")
        
        self.transaction_stack.pop()
    
    @contextmanager
    def transaction(self, name: str, context: str = "AI操作"):
        """
        事务上下文管理器
        
        Usage:
            with undo_guard.transaction("批量重命名") as tx:
                # 执行AI操作
                rename_all_actors()
                # 如果发生异常，自动回滚
        """
        if not self.enabled:
            yield UndoTransaction(name, context)
            return
            
        tx = self.begin_transaction(name, context)
        try:
            yield tx
            self.end_transaction(success=True)
        except Exception as e:
            self.cancel_transaction()
            tx.error = str(e)
            raise
    
    # ========================================================================
    # 常用操作的快捷方法
    # ========================================================================
    
    def wrap_actor_operation(self, name: str, operation: Callable[[], Any]) -> Any:
        """包装Actor操作"""
        with self.transaction(name, "Actor操作"):
            return operation()
    
    def wrap_asset_operation(self, name: str, operation: Callable[[], Any]) -> Any:
        """包装资产操作"""
        with self.transaction(name, "资产操作"):
            return operation()
    
    def wrap_level_operation(self, name: str, operation: Callable[[], Any]) -> Any:
        """包装关卡操作"""
        with self.transaction(name, "关卡操作"):
            return operation()
    
    # ========================================================================
    # 批量操作支持
    # ========================================================================
    
    class BatchOperation:
        """批量操作助手 - 确保全部成功或全部回滚"""
        
        def __init__(self, guard: 'UndoGuard', name: str):
            self.guard = guard
            self.name = name
            self.operations: List[Callable[[], Any]] = []
            self.results: List[Any] = []
            
        def add(self, operation: Callable[[], Any]):
            """添加一个操作"""
            self.operations.append(operation)
            return self
        
        def execute(self) -> List[Any]:
            """执行所有操作"""
            def _batch_op():
                for i, op in enumerate(self.operations):
                    try:
                        result = op()
                        self.results.append(result)
                        unreal.log(f"[Batch] 操作 {i+1}/{len(self.operations)} 成功")
                    except Exception as e:
                        unreal.log_error(f"[Batch] 操作 {i+1} 失败: {e}")
                        raise
                return self.results
            
            return self.guard.wrap_actor_operation(self.name, _batch_op)
    
    def batch(self, name: str) -> 'UndoGuard.BatchOperation':
        """创建批量操作"""
        return self.BatchOperation(self, name)
    
    # ========================================================================
    # 状态管理
    # ========================================================================
    
    def get_stack_info(self) -> List[Dict[str, Any]]:
        """获取事务栈信息"""
        return [tx.to_dict() for tx in self.transaction_stack]
    
    def is_in_transaction(self) -> bool:
        """是否处于事务中"""
        return len(self.transaction_stack) > 0
    
    def get_current_depth(self) -> int:
        """获取当前事务深度"""
        return len(self.transaction_stack)
    
    def enable(self):
        """启用事务保护"""
        self.enabled = True
        unreal.log("[UndoGuard] 事务保护已启用")
    
    def disable(self):
        """禁用事务保护（谨慎使用）"""
        self.enabled = False
        unreal.log_warning("[UndoGuard] 事务保护已禁用")


# 全局实例
_undo_guard_instance: Optional[UndoGuard] = None

def get_undo_guard() -> UndoGuard:
    """获取全局UndoGuard实例"""
    global _undo_guard_instance
    if _undo_guard_instance is None:
        _undo_guard_instance = UndoGuard()
    return _undo_guard_instance


# 装饰器支持
def transactional(name: str = None, context: str = "AI操作"):
    """
    事务装饰器
    
    Usage:
        @transactional("重命名Actor")
        def rename_actor(actor, new_name):
            actor.set_actor_label(new_name)
    """
    def decorator(func: Callable):
        def wrapper(*args, **kwargs):
            tx_name = name or func.__name__
            guard = get_undo_guard()
            with guard.transaction(tx_name, context):
                return func(*args, **kwargs)
        return wrapper
    return decorator
```

## ❓ 常见问题

### Q1: 事务回滚失败怎么办？
A:
1. 检查是否在事务中调用了非回滚操作（如文件IO）
2. 确保所有Actor操作都在事务内进行
3. 查看日志：`Saved/Logs/UndoGuard.log`

### Q2: 如何部分回滚？
A: UndoGuard不支持部分回滚。如需部分回滚，拆分为多个小事务：
```python
guard = UndoGuard()

guard.wrap_actor_operation("第一部分", op1)
if condition:
    guard.wrap_actor_operation("第二部分", op2)  # 这部分可单独撤销
```

### Q3: 事务嵌套太多导致性能下降？
A:
- 减少嵌套层数（最大5层）
- 合并小事务为大事务
- 禁用自动保存（`AutoSaveInterval=0`）
