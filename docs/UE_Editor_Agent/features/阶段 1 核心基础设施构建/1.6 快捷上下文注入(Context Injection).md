# 1.6 快捷上下文注入(Context Injection)

**阶段**：阶段1 - 核心基础设施构建  
**重要性**：⭐⭐⭐⭐ (P1)  
**依赖**：1.5 主线程调度器(GameThread Marshalling)  
**预估工时**：1天

---

## 🎯 目标与里程碑

### 核心目标
构建**上下文注入系统**，自动收集当前编辑器状态（选中对象、当前关卡、项目设置等），为AI提供完整的执行上下文。

### 验收标准
- [ ] 自动收集当前选中Actor列表
- [ ] 获取当前编辑的资产路径
- [ ] 识别编辑器模式（关卡编辑/蓝图/材质）
- [ ] 上下文收集延迟<10ms
- [ ] 支持自定义上下文扩展

---

## 🔧 核心实现

```python
import unreal
from typing import Dict, Any, List, Optional
import json

class ContextInjector:
    """上下文注入器"""
    
    def __init__(self):
        self.context_cache: Dict[str, Any] = {}
        self.cache_ttl = 1.0  # 缓存1秒
        self.last_update = 0.0
    
    def get_full_context(self) -> Dict[str, Any]:
        """获取完整上下文"""
        current_time = time.time()
        if current_time - self.last_update < self.cache_ttl:
            return self.context_cache
        
        context = {
            "selected_actors": self._get_selected_actors(),
            "current_level": self._get_current_level(),
            "current_asset": self._get_current_asset(),
            "editor_mode": self._get_editor_mode(),
            "project_info": self._get_project_info(),
            "viewport_info": self._get_viewport_info()
        }
        
        self.context_cache = context
        self.last_update = current_time
        
        return context
    
    def _get_selected_actors(self) -> List[Dict[str, Any]]:
        """获取选中Actor"""
        actors = unreal.EditorLevelLibrary.get_selected_level_actors()
        return [{
            "name": actor.get_name(),
            "class": actor.get_class().get_name(),
            "path": actor.get_path_name(),
            "location": str(actor.get_actor_location()),
            "rotation": str(actor.get_actor_rotation())
        } for actor in actors]
    
    def _get_current_level(self) -> Dict[str, str]:
        """获取当前关卡"""
        level = unreal.EditorLevelLibrary.get_editor_world()
        if level:
            return {
                "name": level.get_name(),
                "path": level.get_path_name()
            }
        return {}
    
    def _get_current_asset(self) -> Optional[str]:
        """获取当前编辑资产"""
        assets = unreal.EditorUtilityLibrary.get_selected_assets()
        if assets:
            return assets[0].get_path_name()
        return None
    
    def _get_editor_mode(self) -> str:
        """获取编辑器模式"""
        # 简化的模式检测
        if unreal.EditorLevelLibrary.get_selected_level_actors():
            return "level_edit"
        elif unreal.EditorUtilityLibrary.get_selected_blueprint():
            return "blueprint"
        return "unknown"
    
    def _get_project_info(self) -> Dict[str, str]:
        """获取项目信息"""
        return {
            "project_dir": str(unreal.Paths.project_dir()),
            "engine_version": str(unreal.SystemLibrary.get_engine_version()),
            "project_name": unreal.Paths.get_project_file_path()
        }
    
    def _get_viewport_info(self) -> Dict[str, Any]:
        """获取视口信息"""
        # 简化实现
        return {
            "camera_location": "(0,0,0)",
            "camera_rotation": "(0,0,0)"
        }


# 使用示例
if __name__ == "__main__":
    injector = ContextInjector()
    context = injector.get_full_context()
    print(json.dumps(context, indent=2, ensure_ascii=False))
