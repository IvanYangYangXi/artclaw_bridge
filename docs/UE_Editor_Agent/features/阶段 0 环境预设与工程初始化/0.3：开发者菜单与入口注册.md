# 阶段 0.3：开发者菜单与可停靠窗口 (Tab Window) 详细实现

**目标**：在UE编辑器中创建可停靠的Dashboard窗口，并集成到工具栏，实现与UUEAgentSubsystem的状态实时同步。

**🚩 里程碑**：
1. UE编辑器顶部工具栏出现Agent图标按钮
2. 点击按钮弹出可停靠的Dashboard窗口，支持拖拽吸附
3. 窗口实时显示插件版本号、连接状态、统计信息
4. 状态变更时UI自动刷新（无需手动关闭重开）

---

### 1. 完整验证清单

#### 1.1 功能验证
- [ ] **编译通过**：插件无报错编译成功
- [ ] **图标显示**：工具栏出现Agent图标，样式与UE一致
- [ ] **窗口弹出**：点击图标弹出Dashboard窗口
- [ ] **可停靠**：窗口可以拖拽吸附到Content Browser、Details等面板旁
- [ ] **信息展示**：窗口显示版本号、连接状态、统计信息

- [ ] **按钮功能**："Test Connection"按钮能切换状态，"View Logs"能打开Output Log

#### 1.2 高级验证

- [ ] **编辑器重启**：重启UE后，插件自动加载，状态恢复初始值
- [ ] **多窗口**：同时打开多个编辑器窗口时，状态同步正确

### 2. 样式与图标规范

#### 2.1 图标要求
- **格式**：PNG
- **尺寸**：20x20 像素
- **颜色**：遵循UE编辑器配色方案（暗色主题）
- **样式名称**：`UEEditorAgent.PluginAction`

#### 2.2 Slate样式最佳实践
- 使用UE标准的内边距（`Padding(16, 8)`）
- 标题字体使用 `Bold, 18`
- 状态文本使用默认字体，通过颜色区分状态
- 按钮保持水平间距（`Padding(0, 0, 8, 0)`）
- **不使用SSeparator**，通过合理的边距来分割内容
