# 阶段 0.3：开发者菜单与入口注册 (Developer UI Entrance)
我们将使用 C++ 在虚幻引擎顶部的工具栏（Toolbar）或菜单栏中注册一个专属入口。这不仅是为了美观，更是为了给后续的 OpenClaw 连接状态监控 和 对话面板唤起 提供物理挂载点。

**目标**：通过 C++ 扩展编辑器 UI，在顶部工具栏添加 Agent 图标，并绑定基础点击事件。

- [ ] **0.3.1 定义 UI 命令 (UI Commands)**
    - **操作**：继承 `TCommands<FUEEditorAgentCommands>` 类。
    - **细节设置**：定义一个名为 `OpenAgentWindow` 的命令，绑定快捷键（可选，如 `Ctrl+Shift+A`）。
- [ ] **0.3.2 菜单扩展器 (Menu Extender)**
    - **操作**：在 `FUEEditorAgentModule` 模块中使用 `FExtender`。
    - **细节设置**：将入口放入 `Window -> AI Tools` 子菜单下，确保符合 UE 插件设计规范。
- [ ] **0.3.3 工具栏按钮注入 (Toolbar Injection)**
    - **操作**：通过 `UToolMenus` API 在 Level Editor 的工具栏末尾添加图标。
    - **细节设置**：使用自定义图标（.png），并设置 Tooltip 为 "Open UE Editor Agent Panel"。
- [ ] **0.3.4 基础面板占位 (Slate Tab)**
    - **操作**：注册一个 `SDockTab`。
    - **细节设置**：点击菜单后，弹出一个带有 "Agent Server Status: Disconnected" 文字的空白面板。

## 关键代码参考 (C++)

在 `UEEditorAgent.cpp` 的 `StartupModule` 中：

```cpp
void FUEEditorAgentModule::StartupModule()
{
    // 1. 注册命令
    FUEEditorAgentCommands::Register();
    
    // 2. 绑定动作
    PluginCommands = MakeShareable(new FUICommandList);
    PluginCommands->MapAction(
        FUEEditorAgentCommands::Get().OpenAgentWindow,
        FExecuteAction::CreateRaw(this, &FUEEditorAgentModule::OnOpenAgentWindowClicked),
        FCanExecuteAction()
    );

    // 3. 注入工具栏 (UE 5.x 推荐方式)
    UToolMenus::RegisterStartupCallback(FSimpleMulticastDelegate::FDelegate::CreateLambda([this]()
    {
        UToolMenu* ToolbarMenu = UToolMenus::Get()->ExtendMenu("LevelEditor.LevelEditorToolBar.PlayToolBar");
        FToolMenuSection& Section = ToolbarMenu->FindOrAddSection("AgentTools");
        FToolMenuEntry& Entry = Section.AddEntry(FToolMenuEntry::InitToolBarButton(
            FUEEditorAgentCommands::Get().OpenAgentWindow
        ));
    }));
}
```

**🚩 里程碑**：
1. 编译通过后，UE 编辑器顶部工具栏出现一个新的按钮图标。
2. 点击按钮能成功弹出一个名为 "UE Editor Agent" 的可停靠窗口（Dockable Tab）。
3. 窗口内初步显示插件版本号和基础状态信息。
