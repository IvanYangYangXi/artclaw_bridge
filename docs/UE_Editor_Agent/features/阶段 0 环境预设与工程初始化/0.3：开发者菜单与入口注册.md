# 阶段 0.3：开发者菜单与可停靠窗口 (Tab Window) 详细实现

为了实现“可停靠（Dockable）”且能通过工具栏唤起的窗口，我们需要在插件模块中注册一个 **Nomad Tab**。

---

### 1. 核心常量定义 (UEAgentConstants.h)
定义一个全局唯一的 Tab ID，用于在菜单和布局中识别该窗口。
```cpp
#pragma once
#include "CoreMinimal.h"

static const FName UEAgentTabName("UEAgentDashboard");
```

---

### 2. 插件模块逻辑 (FUEAgentModule.cpp)
你需要在 `StartupModule` 中完成两件事：注册 **Tab 产生器 (Tab Spawner)** 和 **工具栏按钮**。

#### 2.1 注册 Tab Spawner
```cpp
#include "Widgets/Docking/SDockTab.h"
#include "Widgets/Text/STextBlock.h"
#include "UEAgentConstants.h"

void FUEAgentModule::StartupModule()
{
    // 1. 注册样式（见上一步实现）
    FUEAgentStyle::Initialize();

    // 2. 注册 Tab 产生器
    FGlobalTabmanager::Get()->RegisterNomadTabSpawner(UEAgentTabName,
        FOnSpawnTab::CreateRaw(this, &FUEAgentModule::OnSpawnPluginTab))
        .SetDisplayName(FText::FromString("UE Editor Agent"))
        .SetMenuType(ETabSpawnerMenuType::Enabled);

    // 3. 注册工具栏按钮 (UToolMenus 逻辑同上)
    UToolMenus::RegisterStartupCallback(FSimpleMulticastDelegate::FDelegate::CreateRaw(this, &FUEAgentModule::RegisterMenus));
}

// 2.2 定义窗口内容 (Slate UI)
TSharedRef<SDockTab> FUEAgentModule::OnSpawnPluginTab(const FSpawnTabArgs& SpawnTabArgs)
{
    auto* Subsystem = GEditor->GetEditorSubsystem<UUEAgentSubsystem>();
    FString ConnectionStatus = (Subsystem && Subsystem->bIsConnected) ? TEXT("Online") : TEXT("Offline");

    return SNew(SDockTab)
        .TabRole(ETabRole::NomadTab)
        [
            SNew(SVerticalBox)
            + SVerticalBox::Slot().AutoHeight().Padding(10)
            [
                SNew(STextBlock)
                .Text(FText::FromString("UE Editor Agent Dashboard"))
                .Font(FCoreStyle::GetDefaultFontStyle("Bold", 18))
            ]
            + SVerticalBox::Slot().AutoHeight().Padding(10, 5)
            [
                // 显示版本号与基础状态
                SNew(STextBlock).Text(FText::FromString("Plugin Version: 1.0.0-alpha"))
            ]
            + SVerticalBox::Slot().AutoHeight().Padding(10, 5)
            [
                SNew(STextBlock).Text(FText::FromString(FString::Printf(TEXT("Status: %s"), *ConnectionStatus)))
            ]
        ];
}
```

#### 2.3 修改工具栏点击回调
修改 `OnClickedAgentButton` 函数，使其不再弹出独立窗口，而是唤起该 Tab。
```cpp
void FUEAgentModule::OnClickedAgentButton()
{
    // 唤起或聚焦到该 Tab
    FGlobalTabmanager::Get()->TryInvokeTab(UEAgentTabName);
}
```

### 4. 遗留问题与优化
目前的 UI 是“静态”的，即：如果你在 Python 里改变了状态，已打开的 Tab 窗口内的文字**不会自动刷新**。

**建议补充逻辑：**
在 `UUEAgentSubsystem` 中定义一个 **`FOnStatusChanged` Delegate**。在 `SDockTab` 创建时绑定该委托，实现 UI 文本的动态更新。
