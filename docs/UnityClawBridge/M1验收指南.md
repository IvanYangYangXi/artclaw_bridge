# UnityClawBridge M1 验收指南

**适用对象**：Unity 新手 | **验收时间**：约 10 分钟

---

## 一、创建 Unity 工程

### 1.1 下载安装 Unity Hub

1. 访问 [unity.com/download](https://unity.com/download)
2. 下载 **Unity Hub**（绿色按钮）
3. 安装并登录 Unity 账号（免费）

### 1.2 安装 Unity 编辑器

1. 打开 Unity Hub → **Installs** → **Add**
2. 选择版本：**Unity 2022.3 LTS**（或更高）
3. 勾选模块：
   - ✅ **Microsoft Visual Studio Community**（代码编辑器）
   - ✅ **WebGL Build Support**（可选）
4. 点击 **Install**

> ⚠️ 第一次安装约 10-20 GB，耐心等待。

### 1.3 创建新工程

1. Unity Hub → **Projects** → **New Project**
2. Template: **3D**（或 **Universal RP**）
3. Project Name: `TestArtClaw`
4. Location: 任意路径
5. 点击 **Create**

---

## 二、安装 UnityClawBridge 插件

### 2.1 方式 A：UPM 包（推荐）

1. 在 Unity 菜单栏：**Window** → **Package Manager**
2. 点击左上角 **+** → **Add package from git URL...**
3. 粘贴插件仓库的 git URL（联系开发者获取）
4. 点击 **Add**，等待安装完成

### 2.2 方式 B：本地安装

1. 将 `subprojects/UnityClawBridge` 文件夹复制到 `Assets/` 下
2. Unity 自动检测并导入
3. 菜单栏出现 **ArtClaw** 菜单

### 2.3 安装 Python 依赖

打开终端，进入 `subprojects/UnityClawBridge/Python/`：

```bash
cd d:/MyProject/artclaw_bridge/subprojects/UnityClawBridge/Python
pip install -r requirements.txt
```

---

## 三、启动 ArtClaw Bridge

### 3.1 启动 Python MCP Server

在终端运行：

```bash
python bootstrap.py --port 8088
```

看到以下输出表示启动成功：

```
[ArtClaw|Unity] INFO 注册工具: run_unity_python
[ArtClaw|Unity] INFO 注册工具: validate_script
[ArtClaw|Unity] INFO 注册工具: unity_health
[ArtClaw|Unity] INFO 注册工具: unity_logs
[ArtClaw|Unity] INFO 已注册 4 个 MCP 工具
[ArtClaw|Unity] INFO MCP Server 启动 ws://127.0.0.1:8088
```

### 3.2 打开 Dashboard

1. 在 Unity 编辑器菜单栏点击 **ArtClaw** → **Dashboard**（快捷键 `Ctrl+Shift+A`）
2. Dashboard 窗口显示：
   - 连接状态（应为"● 运行中"）
   - Unity 版本号
   - MCP 端口 8088

```
Tool: unity_health
```

```
Tool: run_unity_python
Arguments: {"code": "Debug.Log(\"Hello from ArtClaw!\");"}
Expected:
  - Unity Console 显示 "[ArtClaw|AI] Hello from ArtClaw!"
  - 返回 "OK"
```

#### Test 3：获取场景信息
```
Tool: run_unity_python
Arguments: {"code": "Scene.activeScene.name"}
Expected: 返回当前场景名称（如 "SampleScene"）
```

#### Test 4：查找 GameObject
```
Tool: run_unity_python
Arguments: {"code": "GameObject.Find(\"Main Camera\").name"}
```

```
Tool: validate_script
```

## 五、常见问题

### Q1：Dashboard 显示"已停止"
- 确保 Python MCP Server 正在运行（终端窗口未关闭）
- 确保端口 8088 未被占用

### Q2：Python 报错 "No module named 'websockets'"
```bash
pip install websockets aiohttp
```

### Q3：Unity Console 无输出
- 检查 Unity 版本 ≥ 2022.3
- 检查 Console 面板是否展开

### Q4：连接 MCP 失败
- 确认 AI 客户端配置了 `127.0.0.1:8088`
- 检查防火墙未阻止本地连接
