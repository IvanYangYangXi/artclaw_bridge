# UE AI助手插件架构设计

## 1. 项目概述

### 1.1 项目目标
开发一个集成到Unreal Engine编辑器的AI助手插件，通过自然语言交互实现对UE编辑器的智能控制。

### 1.2 核心功能
1. **自然语言聊天界面** - UE编辑器内的对话式交互界面
2. **LLM API集成** - 连接多种大语言模型API
3. **实时双向通信** - WebSocket长连接支持
4. **任务执行引擎** - 将自然语言转换为UE操作
5. **蓝图扩展** - 暴露AI功能给蓝图系统

### 2.2 技术栈
- **引擎**: Unreal Engine 5.3+
- **语言**: C++ 17/20 + Blueprint
- **UI框架**: Slate + UMG
- **网络**: HTTP/WebSocket
- **JSON**: RapidJSON/UE JSON Utilities
- **并发**: AsyncTask + TaskGraph
